# 物品管理系统 📦  

本项目用于管理自己拥有的物品，并计算每日均价。数据将自动存储在 SQLite3 数据库中。  

## 🚀 快速启动  

确保您的系统已安装 **Python 3**，然后运行app.py即可。 (一些库pip安装即可)
